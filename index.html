Faux-Par

Many programs require a series of key/value to define their configuration and behavior. For command line programs, these key/value parameters are often read from a text file. It would be so cool, if a tool existed, that generates a GUI for such parameters automatically. Faux-Par tries to accomplish this.

Process

The C++ program needs to define its required parameters by creating FauxPar objects. In fact, each class can define its own parameters. It implements the static method `FauxPar::FauxPar ExampleClass1::GetFauxPar()`. Here's an example for a float value:

FauxPar::FauxPar ExampleClass1::GetFauxPar() { FauxPar::FauxPar parameters("Example1"); parameters.insert(new FauxPar::ValueFloat("speed")); return parameters; }

Creating pages manually
